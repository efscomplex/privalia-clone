# React Shopping Cart
This is a react shopping cart app not a functional but only for demostration purposes. 
The app is bootstraped with a create react app templated with typescript.

## `Stack`
- React library
- Styles
   - styled components
   - module imports
- Store
   - Based in redux flow
   - customized with a provider hook
